PEAR Channel for the PHP Quality Assurance Toolchain

Registering the channel: 
pear channel-discover pear.phpqatools.org
Listing available packages: 
pear remote-list -c phpqatools
Installing a package: 
pear install phpqatools/package_name
Installing a specific version/stability: 
pear install phpqatools/package_name-1.0.0
pear install phpqatools/package_name-beta
Receiving updates via a feed: 
http://pear.phpqatools.org/feed.xml
